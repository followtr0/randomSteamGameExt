# Chrome Web Store Assets

Assets needed for Chrome Web Store listing.

## Required:

### 1. Screenshots (1280x800 or 640x400)
Place at least 1 screenshot here showing:
- The circular 🎮 button on Steam library page
- The button expanded on hover showing "Get random game!"
- The modal showing a selected game
- The "looking for a game" animation (optional)

**Files:** `screenshot1.png`, `screenshot2.png`, etc.

### 2. Store Icon
Already covered by `icons/icon128.png`

## Optional (but increases visibility):

### 3. Small Promotional Tile (440x280)
Shows in search results and category pages
**File:** `promo-small.png`

### 4. Marquee Promotional Tile (1400x560)
For featured placements (rare, but good to have)
**File:** `promo-marquee.png`

## Tips for Screenshots:
- Use Steam's dark theme for consistency
- Show the extension in action
- Highlight the key features (button, modal, game selection)
- Use annotations/arrows if helpful
- Make sure text is readable

## Screenshot Ideas:
1. Steam library with button visible (default state)
2. Button hovered showing text
3. Modal with "looking for a game"
4. Modal showing selected game with details
5. Different game showing achievements/playtime
