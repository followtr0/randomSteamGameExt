# Random Steam Game - Privacy Policy

Last updated: 2026-02-10

Random Steam Game does not collect, store, or share personal data.

## What this extension accesses

- Steam Community library pages (`https://steamcommunity.com/*`) to read the visible game list and pick a random game.
- Browser tabs permission only to open the Steam library page when the user clicks the extension toolbar icon.

## What this extension does not do

- No account creation
- No analytics
- No tracking
- No cookies
- No transmission of personal data to external servers
- No sale or sharing of user data

## Data storage

All extension behavior runs locally in the browser. No personal data is stored remotely.
