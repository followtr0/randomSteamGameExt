# Icon Requirements

Create PNG icons in the following sizes:

## Required Sizes:
- **icon16.png** - 16x16 pixels
- **icon32.png** - 32x32 pixels
- **icon48.png** - 48x48 pixels
- **icon128.png** - 128x128 pixels

## Design Guidelines:
- Use transparent background
- PNG format
- Simple, recognizable design
- Works well at small sizes (16px must be clear)
- Suggested theme: Game controller, dice, shuffle symbol, or Steam logo derivative

## Chrome Web Store Additional Assets:
(Place these in a separate `store-assets` folder)

### Required:
1. **Store Icon** - 128x128 PNG (can reuse icon128.png)
2. **Screenshots** - At least 1, max 5
   - 1280x800 or 640x400 pixels
   - Show extension in action on Steam library page
   - Include the circular button and modal

### Optional but Recommended:
3. **Small tile** - 440x280 PNG
4. **Large tile** - 1400x560 PNG

## Tools:
- Create master at 512x512 or 1024x1024
- Scale down using:
  - Photoshop/GIMP
  - Online: https://www.iloveimg.com/resize-image
  - Command line: ImageMagick
